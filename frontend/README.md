# 🌸 Skin AI Specialist
AI 기반 개인 맞춤형 피부 진단 및 토탈 케어 솔루션

## 🛠 Tech Stack
- **Frontend:** React, Vite, Lucide-React, Axios
- **Backend:** FastAPI, PyTorch, Pandas
- **AI Model:** ResNet18 (Regression)

## 🚀 시작하기
1. **Backend 실행**
   - `cd backend`
   - `python -m uvicorn main:app --reload`
2. **Frontend 실행**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
